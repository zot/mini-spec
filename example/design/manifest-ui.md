# UI Manifest

## Theme

**Light Mode**
- Background: #ffffff
- Surface: #f5f5f5
- Text: #333333
- Text secondary: #666666
- Primary: #0066cc
- Danger: #cc3333
- Border: #dddddd

**Dark Mode** (class: .dark)
- Background: #1a1a1a
- Surface: #2d2d2d
- Text: #eeeeee
- Text secondary: #aaaaaa
- Primary: #4da6ff
- Danger: #ff6666
- Border: #444444

## Global Components

**Buttons**
- Padding: 8px 16px
- Border radius: 4px
- Primary: filled with primary color
- Secondary: outlined
- Danger: filled with danger color

**Inputs**
- Padding: 8px
- Border: 1px solid border color
- Border radius: 4px
- Focus: primary color border

**List Items**
- Padding: 12px
- Hover: surface color background
- Selected: primary color with light opacity

## Responsive

- Min width: 600px
- List/detail split: 30%/70%
- Stack vertically below 600px (future)
